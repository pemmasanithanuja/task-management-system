# 📋 Task Management System

A web-based **Task Management System** developed using **Python** and **Flask** that helps users efficiently organize, manage, and track their daily tasks through a simple and intuitive interface.

---

## 🚀 Project Overview

This application provides an easy way to manage personal or professional tasks. Users can create, view, update, and delete tasks, making task organization more efficient and improving productivity.

---

## ✨ Features

- ➕ Add new tasks
- 📄 View all tasks
- ✏️ Update existing tasks
- 🗑️ Delete completed or unwanted tasks
- 💻 Clean and user-friendly interface
- ⚡ Fast and lightweight application
- 💾 Data stored using SQLite database

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/pemmasanithanuja/task-management-system.git
```

### Move into the project folder

```bash
cd task-management-system
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open in your browser

```
http://127.0.0.1:5000/
```
